# Project Dependencies:

## Dependencies for Running the App:

* TK-Interface (`tkinter`)

```sh
sudo apt-get install python3-tk
```

* `customtkinter`

```sh
pip install customtkinter
```

* `packaging`

```sh
pip install packaging
```

* Pillow (`PIL`)

```sh
pip install Pillow
```

* Requests

```sh
pip install requests
```

* CTkListbox

```sh
pip install CTkListbox
```
