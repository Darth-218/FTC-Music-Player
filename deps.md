# Project Dependencies:

## Dependencies for Running the App:

* `packaging`

```sh
pip install packaging
```

* Pillow (`PIL`)

```sh
pip install Pillow
```

* Requests

```sh
pip install requests
```

* PythonNet

```sh
pip install pythonnet
```

* Python-VLC

```sh
pip install python-vlc
```

* Flet

```sh
pip install flet
```
